# 社会学与组织管理（Sociology and Organization Studies）

本目录从人类社会的协作形态出发，审视多智能体系统（Multi-Agent Systems）的组织、协调与治理机制。

> 社会学的核心洞见在于：**个体理性不等于集体最优，秩序不是设计出来的就是涌现出来的**。
