# 世界模型

本目录整理世界模型的相关研究。

## 什么是世界模型？

世界模型作为环境的内部模拟器，使智能系统能够：
- 预测未来状态
- 理解因果关系
- 在不依赖真实交互的情况下做出决策

## 四大架构类别

### 1. 潜空间模型（Latent Space Models）

在压缩的潜在空间中建模世界动态。

| 模型 | 年份 | 贡献 |
|------|------|------|
| PlaNet | 2019 | 最早采用RSSM架构 |
| DreamerV1 | 2020 | 引入循环状态空间模型 |
| DreamerV2 | 2021 | 离散潜在变量，Atari达到人类水平 |
| DreamerV3 | 2023 | 首个通用世界模型，广泛任务顶尖性能 |
| DayDreamer | 2023 | 物理机器人部署 |
| TransDreamer | 2023 | Transformer替代RNN |

**优势**: 计算高效、成熟稳定
**劣势**: 长时预测能力有限

### 2. JEPA架构

联合嵌入预测架构，通过预测潜在表示来学习。

| 模型 | 特点 |
|------|------|
| I-JEPA | 避免重建计算，表示学习更纯粹 |

**优势**: 表示学习高效
**劣势**: 理论探索阶段，应用较少

### 3. Transformer世界模型

利用注意力机制建模长程依赖。

| 模型 | 特点 |
|------|------|
| Genie | 空间-时间Transformer，无标注视频自监督学习 |
| TWM | Transformer-XL，段级循环机制 |
| iVideoGPT | VQ-VAE多模态序列token化 |

**优势**: 长程依赖建模强
**劣势**: 计算资源需求大

### 4. 扩散世界模型

使用扩散过程生成未来状态。

| 模型 | 特点 |
|------|------|
| Sora | 照片级真实感，工业级视频生成 |
| Cosmos | 层次化扩散渲染，实时文本编辑 |
| RoboDreamer | 零样本组合视频生成 |
| Pandora | 扩散帧合成+自回归骨干 |
| Occ-Sora | 4D占据生成模型 |

**优势**: 生成质量最高
**劣势**: 推理慢，难以实时交互

## 架构对比

| 维度 | 潜空间模型 | JEPA | Transformer WM | 扩散WM |
|------|-----------|------|----------------|--------|
| 计算效率 | 高 | 高 | 中 | 低 |
| 生成质量 | 中 | 中 | 中高 | 最高 |
| 长程建模 | 弱 | 中 | 强 | 中 |
| 实时性 | 好 | 好 | 中 | 差 |
| 成熟度 | 高 | 低 | 中 | 高 |

## 在具身AI中的应用

### 1. 规划验证

- 执行前验证计划可行性
- 预测潜在失败模式
- 想象rollout评估策略

### 2. 数据生成

| 应用 | 描述 |
|------|------|
| 轨迹生成 | 合成机器人操作轨迹 |
| 场景生成 | 多样化训练场景 |
| Corner Case | 罕见边缘情况数据 |
| 视频合成 | 高质量操作视频 |

### 3. 基于模型的强化学习（MBRL）

世界模型作为动力学模型：
- 无需手工设计物理引擎
- 想象rollout模拟未来
- 降低样本复杂度

### 4. 代表性应用

| 方法 | 应用领域 |
|------|----------|
| SWIM | 机器人操作，视频预训练 |
| RoboTransfer | 视觉策略迁移 |
| 盘古世界模型 | 自动驾驶 |

## 发展趋势

**主流方向**: 扩散世界模型 + Transformer融合

| 趋势 | 描述 | 代表工作 |
|------|------|----------|
| 架构融合 | 扩散保质量，Transformer保一致性 | Sora, Cosmos, Genie |
| 大规模预训练 | 互联网视频数据预训练 | Sora, SWIM |
| 实时交互改进 | 解决推理慢问题 | Pandora, Cosmos |
| 潜在扩散 | 兼顾效率和质量 | Latent Diffusion |

## 挑战与未来方向

### 当前挑战

- **长时预测一致性**: 保持长时间预测一致性
- **物理规律建模**: 准确建模复杂物理现象
- **计算效率**: 大模型推理延迟
- **真实世界验证**: 仿真到真实迁移

### 未来方向

- 更强的物理常识建模
- 与LLM/VLM深度集成
- 实时交互式世界模型
- 跨域泛化能力

## 应用场景推荐
