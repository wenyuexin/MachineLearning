## 一、论文整体思路

### 1.1 研究背景

具身智能（Embodied AI）的探索已跨越数十年，但赋予智能体人类水平的智能（包括感知、学习、推理、决策、控制和泛化能力）以在开放、非结构化、动态环境中执行通用任务仍然是一个持久挑战。

近年来，计算机视觉、自然语言处理和多模态学习的进展表明，基础模型（Foundation Models）在特定任务上展现出超人类能力。它们不仅为将基础模块集成到具身智能系统中提供了坚实基石，也为如何从方法论角度扩展机器人学习提供了启示。

### 1.2 综述定位与贡献

本综述旨在提供基础模型在机器人学中的全面且最新的概览，聚焦于自主操作，涵盖高层规划和底层控制。主要贡献包括：

1. **系统分类**：基于高层规划策略和底层控制机制对现有文献进行分类
2. **资源汇总**：广泛调查机器人、仿真器和基准测试
3. **挑战与展望**：讨论关键挑战并勾勒有前景的研究方向

## 二、基础模型在机器人学中的应用

### 2.1 基础模型类型

| 模型类型 | 代表模型 | 在机器人学中的作用 |
|----------|----------|-------------------|
| **大语言模型 (LLM)** | GPT系列、LLaMA | 任务规划、推理、代码生成 |
| **视觉基础模型** | CLIP、SAM、DINO | 视觉感知、特征提取 |
| **多模态模型** | GPT-4V、Gemini | 跨模态理解、视觉问答 |
| **机器人基础模型** | RT-1、RT-2、RoboCat | 端到端控制 |

### 2.2 应用层次

```
高层规划 (High-Level Planning)
    ↓
中间表示 (Intermediate Representation)
    ↓
底层控制 (Low-Level Control)
```

## 三、高层规划

### 3.1 基于LLM的任务规划

**核心思想**：利用LLM的语义理解和推理能力进行任务分解和规划。

| 方法类别 | 描述 | 代表工作 |
|----------|------|----------|
| **任务分解** | 将复杂任务分解为子任务 | SayCan, LLM-Planner |
| **规划生成** | 生成可执行的动作序列 | Progprompt, Code as Policies |
| **约束满足** | 考虑环境约束的规划 | LLM+P, REFLECT |

**关键技术**：
- 零样本/少样本任务分解
- 上下文学习（In-context Learning）
- 思维链（Chain-of-Thought）推理

### 3.2 基于视觉语言模型的规划

**核心思想**：结合视觉感知和语言理解进行规划。

| 能力 | 描述 | 应用 |
|------|------|------|
| 视觉问答 | 理解场景内容 | 目标识别、状态估计 |
| 视觉 grounding | 将语言与视觉对齐 | 物体定位、操作点检测 |
| 视觉推理 | 基于视觉的逻辑推理 | 任务规划、异常检测 |

### 3.3 代码生成规划

**核心思想**：将自然语言指令转换为可执行代码。

| 方法 | 描述 |
|------|------|
| Code as Policies | 使用LLM生成Python策略代码 |
| Progprompt | 程序化提示生成可执行策略 |
| VoxPoser | 基于视觉语言模型生成3D价值图 |

## 四、底层控制

### 4.1 模仿学习

**核心思想**：从专家演示中学习控制策略。

| 方法 | 描述 | 代表工作 |
|------|------|----------|
| 行为克隆 | 直接回归动作 | 经典方法 |
| ACT | 动作块Transformer | Action Chunking Transformer |
| 扩散策略 | 扩散模型生成动作 | Diffusion Policy |

**数据类型**：
- 人类遥操作数据
- 动捕数据
- 仿真生成数据

### 4.2 强化学习

**核心思想**：通过与环境交互试错学习最优策略。

| 方法 | 描述 | 代表工作 |
|------|------|----------|
| 无模型RL | 直接学习策略/价值函数 | PPO, SAC |
| 基于模型的RL | 学习世界模型 | Dreamer, TD-MPC |
| 离线RL | 从固定数据集学习 | CQL, IQL |

**挑战**：
- 样本效率低
- 真实环境训练不安全
- 奖励函数设计困难

### 4.3 预训练与迁移

| 方法 | 描述 | 代表工作 |
|------|------|----------|
| 视觉预训练 | 使用互联网数据预训练视觉编码器 | CLIP, R3M |
| 策略预训练 | 在大规模机器人数据上预训练 | RT-1, Octo |
| 跨任务迁移 | 将学到的技能迁移到新任务 | 各种微调方法 |

## 五、数据集与基准测试

### 5.1 机器人数据集

| 数据集 | 规模 | 特点 |
|--------|------|------|
| **Open X-Embodiment** | 1M+轨迹 | 跨机器人、跨任务 |
| **BridgeData** | 数十K | 真实机器人操作 |
| **RLBench** | 100+任务 | 仿真任务多样性 |
| **CALVIN** | 长期任务 | 语言条件操作 |

### 5.2 仿真器

| **Habitat** | 导航任务 | SPL、成功率 |

## 六、当前挑战

### 6.1 技术挑战

1. **泛化能力**
   - 跨物体泛化
   - 跨场景泛化
   - 跨任务泛化

2. **样本效率**
   - 减少真实环境交互
   - 提高数据利用率
   - 利用人类视频数据

3. **实时性**
   - 大模型推理延迟
   - 控制频率要求
   - 边缘部署

### 6.2 系统挑战

1. **Sim-to-Real差距**
   - 视觉域差距
   - 物理动力学差异
   - 系统延迟

2. **安全性**
   - 真实环境交互风险
   - 不确定性量化
   - 安全约束学习

3. **可复现性**
   - 硬件依赖
   - 环境差异
   - 随机性控制

## 八、关键见解与总结

### 8.1 核心贡献

本综述提供了基础模型在机器人学中应用的全面概览：
- 系统分类：高层规划和底层控制
- 资源汇总：数据集、仿真器、基准测试
- 挑战分析：技术和系统层面的挑战

### 8.2 技术趋势

1. **大模型驱动**：LLM和VLM正成为机器人系统的核心组件
2. **端到端学习**：从模块化向端到端架构演进
3. **数据驱动**：利用大规模数据预训练通用策略
4. **Sim-to-Real**：缩小仿真与真实的差距

### 8.3 开放问题

- 如何构建真正通用的机器人基础模型？
- 如何高效利用多样化数据源？
- 如何确保真实环境部署的安全性？
- 如何评估和比较不同方法？
