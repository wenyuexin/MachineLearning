## 一、论文整体思路

### 1.1 研究背景

移动操作需要机器人能够快速学习新任务，传统方法需要大量演示。One-shot学习可以大幅降低数据需求。

**核心问题**：
- 如何从单个人类演示快速学习？
- 如何迁移到未见环境？
- 如何处理移动基座的运动学差异？

### 1.2 核心创新

HRT1实现**从单个演示 → 移动操作 → 真实世界执行**

**核心特点**：
- AR头显指导的演示
- 3D感知与抓取迁移
- 约束优化的轨迹求解
- 鲁棒的实时执行

## 三、关键技术

### 3.1 AR引导演示

**AR头显作用**：
- 提供机器人视角的演示环境
- 增强现实叠加目标物体
- 实时反馈和指导

**优势**：
- 演示与执行视角一致
- 减少视角转换误差
- 提升迁移准确性

### 3.2 3D感知与抓取

**感知流程**：
```
AR演示视频 → 物体检测 → 3D位姿估计 → 
匹配真实环境 → 抓取策略生成
```

**关键能力**：
- 物体识别与定位
- 3D场景重建
- 跨环境物体匹配

### 3.3 轨迹迁移

**迁移策略**：
```
人手轨迹(3D) → 重定向到机器人基座 → 
物体-中心对齐 → 机器人末端轨迹
```

**对齐方法**：
- 物体相对坐标
- 末端执行器姿态保持
- 速度与加速度适配

### 3.4 约束优化

**优化目标**：
- 最小化轨迹误差
- 满足关节限制
- 避免碰撞
- 保持可达性

**求解器**：
- 逆运动学(IK)
- 非线性优化
- 实时约束处理

## 四、实验与结果

### 4.1 任务设置

**移动操作任务**：
- 桌面物体抓取
- 货架物品取放
- 开门操作
- 拉抽屉

**机器人**：
- 移动基座 + 操作臂
- 6-DOF机械臂
- 灵巧夹爪

### 4.2 One-Shot学习性能

| 任务 | 成功率 | 备注 |
|------|--------|------|
| 桌面抓取 | 高 | 单次演示即可 |
| 货架取放 | 中高 | 需要可达性验证 |
| 开门操作 | 中 | 需要轨迹优化 |
| 拉抽屉 | 中高 | 物体-中心对齐有效 |

### 4.3 鲁棒性验证

**场景变化测试**：
- 不同光照条件
- 物体位置偏移
- 障碍物存在

**结果**：
- 对适度场景变化鲁棒
- 物体-中心对齐关键
- 约束优化避免碰撞

### 4.4 与传统方法对比

| 方法 | 所需演示数 | 成功率 | 训练时间 |
|------|------------|--------|---------|
| 传统IL | 10-50 | 高 | 长 |
| One-Shot方法 | 1 | 中-高 | 无需训练 |
| HRT1 | 1 | 中-高 | 无需训练 |

## 六、技术细节

### 6.1 AR头显

**功能**：
- 显示机器人视角
- 叠加3D信息
- 引导操作演示

**优势**：
- 减少视角差异
- 提升迁移准确性
- 实时反馈

### 6.2 3D感知

**感知 pipeline**：
```
RGB-D输入 → 物体检测 → 3D位姿估计 → 
场景理解 → 物体匹配
```

**关键算法**：
- 深度学习物体检测
- 位姿估计网络
- 点云匹配

### 6.3 轨迹优化

**优化变量**：
- 关节角度序列
- 基座运动轨迹
- 末端执行器姿态

**约束**：
- 关节限制
- 可达性
- 避障
- 平滑性
