## 一、论文整体思路

### 1.1 研究背景

视觉-语言-动作（VLA）模型在机器人操作中展现出巨大潜力，但机器人演示数据稀缺限制了其发展。人类视频提供了大量操作演示，但现有方法存在关键问题。

**核心问题**：
- 机器人演示数据稀缺
- 现有潜在动作模型存在**视觉纠缠（Visual Entanglement）**问题
- 学习的表示包含视觉噪声而非纯粹的操作技能
- 人类视频与机器人动作空间对齐困难

### 1.2 核心创新

CLAP提出**对比潜在动作预训练**框架，显式对齐人类视觉转换与机器人动作的潜在空间。

**核心特点**：
- 解决视觉纠缠问题
- 对比学习对齐潜在空间
- 双公式VLA框架（CLAP-NTP + CLAP-RF）
- 知识匹配正则化策略

### 2.2 对比潜在动作预训练

**核心思想**：通过对比学习将人类视觉转换映射到物理可执行的潜在动作空间

| 组件 | 描述 |
|------|------|
| 视觉转换编码器 | 从人类视频中提取视觉变化特征 |
| 动作量化码本 | 将机器人动作量化为离散token |
| 对比损失 | 拉近相似转换-动作对，推远不相似对 |
| 同构映射 | 确保视觉转换映射到物理控制空间 |

### 2.3 双公式VLA框架

**CLAP-NTP（自回归规划器）**：
- 基于Next Token Prediction
- 擅长语义理解和指令跟随
- 适合高层任务规划

**CLAP-RF（Rectified Flow控制器）**：
- 基于流匹配的连续动作生成
- 高频精确操作能力
- 适合接触丰富的精细操作

### 2.4 知识匹配正则化

解决微调过程中的灾难性遗忘问题：
- 作为正则化项
- 将策略更新锚定在预训练模型的可信区域
- 保留语义知识的同时适应特定任务

## 三、实验与结果

### 3.1 真实世界双臂任务

| 任务 | CLAP-NTP | CLAP-RF | π₀基线 | π₀.₅基线 |
|------|----------|---------|--------|----------|
| 语义理解任务 | 高 | 中 | 高 | 高 |
| 指令跟随任务 | 高 | 中 | 高 | 高 |
| 精细操作任务 | 中 | 高 | 中 | 中 |
| Make Bouquets | 40% | 40% | 40% | 40% |

### 3.2 LIBERO仿真基准

在LIBERO四个任务套件上的表现：
- 平均成功率：**90.3%**
- FM变体：**94.5%**
- 显著超越现有VLA和模仿学习基线

### 3.3 泛化能力

- **对象泛化**：通过人类视觉先验迁移实现强泛化
- **指令跟随**：CLAP-NTP在语言条件任务中表现优异
- **精细控制**：CLAP-RF在接触丰富任务中提供必要精度

## 四、关键贡献

### 4.1 解决视觉纠缠

**问题识别**：现有潜在动作模型学习的表示包含无关视觉噪声
**解决方案**：对比学习显式对齐视觉转换与物理动作

### 4.2 双公式框架

- CLAP-NTP：擅长语义推理和指令跟随
- CLAP-RF：提供高频精确操作能力
- 互补架构覆盖不同任务需求

### 4.3 知识匹配正则化

- 防止微调时灾难性遗忘
- 保留预训练语义知识
- 平衡适应性与知识保持

---

## 五、技术细节

### 5.1 潜在动作空间

```
人类视觉转换 → 视觉编码器 → 潜在动作向量
                              ↕ (对比学习对齐)
机器人动作 → 动作量化器 → 离散动作token
```

### 5.2 训练流程

1. **预训练阶段**：大规模人类视频 + 对比学习
2. **中间训练**：少量机器人数据适配
3. **微调阶段**：任务特定优化 + 知识匹配正则化

## 七、局限性与未来方向

### 7.1 局限性

1. 需要配对的视觉-动作数据进行预训练
2. 双公式架构增加系统复杂度
3. 对极端新颖任务的泛化有限

### 7.2 未来方向

1. 更大规模人类视频预训练
2. 单一统一架构
3. 在线适应学习
4. 跨形态扩展
