# 00-interview (面试题)

本目录汇总各方向常见的面试题目，供复习与自查。

## 目录结构

```
├── deep-learning/   深度学习（基础、网络结构、训练优化、模型评估）
├── llm/             大语言模型（预训练、微调对齐、推理部署、评测）
├── agentic/         智能体（工具调用、多Agent、记忆、RAG、安全）
└── README.md           本文件
```

## 使用方式

每个子目录的 README 按主题列出常见问题与追问方向，适合：

- 面试前快速扫盲
- 按主题自查薄弱环节
- 作为面试官设计题目的参考
