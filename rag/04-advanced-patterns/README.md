# Advanced Patterns

通用 RAG 高级范式，不绑定特定项目或实现。

## 定位

本目录收录 RAG 的通用增强模式与架构范式，聚焦"是什么"和"为什么"，不展开具体项目的实现细节。具体项目笔记请移步 `../05-implementations/`。

## 子目录

| 目录 | 说明 |
|------|------|
| agentic-rag/ | Agent 驱动的 RAG，自主规划与工具调用 |
| modular-rag/ | 模块化 RAG，组件解耦与灵活组合 |
| multimodal-rag/ | 多模态 RAG，跨模态检索与生成 |
| self-reflective-rag/ | 自反思 RAG，检索-生成-修正闭环 |
