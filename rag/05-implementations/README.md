# Implementations

具体 RAG 项目与方案的集中存放区。

## 定位

本目录按项目名组织，每个子目录对应一个具体的 RAG 开源项目或重要方案。内容覆盖项目核心思想、架构设计、关键算法、使用方式及与同类项目的对比。

## 存放原则

- 一个项目一个目录，不混放
- 目录名使用项目的小写短名（如 `graph-rag`, `hipporag`）
- 项目相关的论文笔记、代码分析、面试题统一放在对应目录下

## 现有项目

| 项目 | 目录 | 核心特点 |
|------|------|---------|
| Microsoft GraphRAG | `graph-rag/` | 基于知识图谱的子图检索增强 |
| HippoRAG | `hipporag/` | 海马体记忆索引 + PPR 多跳推理 (NeurIPS 2024) |
