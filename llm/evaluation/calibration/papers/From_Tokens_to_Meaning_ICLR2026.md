## 一、论文整体思路

### 1.1 核心论点

**Token级不确定性对LLM不可靠，语义级不确定性才是评估模型可靠性的正确视角。**

传统方法在封闭世界（closed-world）中有效——输出空间有限且预定义，Token概率能反映真实置信度。但LLM运行在**开放世界**（open-world）中：输出空间无界、查询可能涉及未见实体、存在歧义或需要复杂推理，Token级置信度会严重误导。

### 1.2 核心示例

```
问题: "瑞士的首都是哪里？"
模型输出: "我不太确定，但我认为是苏黎世。"

Token级分析：低熵（模型对每个Token的选择很确定）
语义级分析：模型实际上表达了不确定——"我不太确定"
```

**错位**：Token级低熵 ≠ 语义级确定。模型在Token选择上很自信，但在语义上表达了不确定性。

### 1.3 主要贡献

1. **论证语义级不确定性的必要性**：系统性论述Token级不确定性的根本缺陷
2. **超越幻觉框架**：将LLM可靠性从"是否幻觉"扩展到更广泛的不确定性量化
3. **多模态扩展**：将论证延伸到LVLM（视觉-语言模型），多模态输入引入额外不确定性源
4. **社区呼吁**：推动领域从Token级评估转向语义级评估

## 二、Token级不确定性的根本缺陷

### 2.1 封闭世界 vs 开放世界

| 维度 | 封闭世界（传统NLP） | 开放世界（LLM） |
|------|-------------------|----------------|
| 输出空间 | 有限且预定义 | 无界、开放式 |
| Token概率 | 能反映真实置信度 | 可能严重误导 |
| 语义等价 | 不存在（标签固定） | 大量存在 |
| 不确定性来源 | 主要是预测不确定性 | 输入/推理/参数/预测多维度 |

### 2.2 Token级置信度失效的场景

| 场景 | Token级表现 | 语义级真实情况 | 失效原因 |
|------|-----------|--------------|---------|
| 模型表达"不确定" | 低熵（Token选择确定） | 语义上不确定 | Token概率无法捕捉语用含义 |
| 同义改写 | 高熵（多种表述方式） | 语义上确定 | 混淆了表达多样性与语义不确定性 |
| 幻觉 | 低熵（高置信生成） | 语义上错误 | 高概率Token序列≠正确语义 |
| 多步推理错误 | 中间步骤低熵 | 推理链已断裂 | 局部Token概率不反映全局正确性 |

### 2.3 根本原因

Token级不确定性衡量的是"**选择哪个Token**"的不确定性，而LLM的可靠性取决于"**表达了什么含义**"的不确定性。两者之间存在根本性的语义鸿沟（Semantic Gap）：

```
Token空间的不确定性  ←──语义鸿沟──→  语义空间的不确定性
    ↓                                    ↓
衡量Token选择的一致性              衡量输出含义的确定性
无法捕捉语用含义                  能反映模型对答案的真实把握
对同义改写产生虚假高不确定性       正确识别语义等价
```

## 三、语义级不确定性框架

### 3.0 正式定义

**定义 2.1 Token级不确定性（Token-level Uncertainty）**

Token级不确定性指与单个输出单元（如语言模型中的词或视觉模型中的图像块）相关的不确定性。这类不确定性与深度学习中经典的不确定性概念紧密对齐，预测在细粒度层面进行，通过概率输出建模。

**定义 2.2 语义级不确定性（Semantic-level Uncertainty）**

语义级不确定性指模型对**生成内容含义**的不确定性，而非对特定Token序列的不确定性。它捕捉的是模型对"表达了什么"而非"如何表达"的不确定性。

### 3.1 三步框架

```mermaid
flowchart LR
    A[生成：从模型采样<br/>多个输出序列] --> B[聚类：按语义含义<br/>双向蕴含分组]
    B --> C[熵估计：计算<br/>语义熵]
```

**Step 1 — 生成（Generation）**：从模型预测分布中采样多个输出序列

**Step 2 — 语义聚类（Clustering）**：使用双向蕴含（Bidirectional Entailment）将语义相同的序列聚类

**Step 3 — 语义熵估计（Entropy Estimation）**：对语义聚类计算熵

$$SE = -\sum_{c=1}^{C} p(c) \log p(c)$$

### 3.2 与Token级不确定性的对比

| 维度 | Token级不确定性 | 语义级不确定性 |
|------|---------------|--------------|
| 衡量对象 | Token选择的一致性 | 输出含义的确定性 |
| 对同义改写 | 虚假高不确定性 | 正确识别等价 |
| 对"不确定"表达 | 误判为低不确定性 | 识别语用含义 |
| 计算成本 | 低（直接用logits） | 高（需多次生成+蕴含判断） |
| 适用场景 | 封闭世界分类 | 开放世界生成 |

### 3.3 Token级 vs 语义级不确定性：实验对比

**实验设置**：每问题查询20次，比较两种语义聚类策略（GPT-4o vs DeBERTa）。

**评估指标**：
- ECE（期望校准误差）：越低越好
- AUROC：越高越好

**主要结果**：

| 方法 | ECE (%) | AUROC (%) |
|------|---------|-----------|
| Perplexity（Token级） | **31.50** | 54.07 |
| Semantic Entropy (DeBERTa) | 22.66 | 79.16 |
| Semantic Entropy (GPT-4o) | **13.53** | **80.94** |

**关键发现**：
1. **语义熵AUROC提升26个百分点**：从54%到80%
2. **语义熵ECE降低57%**：从31.50%到13.53%
3. **GPT-4o聚类效果略优于DeBERTa**：可能因更强的语义理解能力

### 3.4 相关方法演进

```
Token级方法
├── 最大Token概率
├── Token熵
└── 序列概率（长度归一化）
    ↓ 语义鸿沟问题
语义级方法
├── 语义熵（Semantic Entropy, Kuhn et al. ICLR 2023）
│   └── NLI蕴含聚类 + 熵计算
├── 核语言熵（Kernel Language Entropy, NeurIPS 2024）
│   └── 正定核编码语义依赖 + von Neumann熵
└── 语义熵探测器（SEPs）
    └── 高效近似，降低5-10倍成本
```

## 四、多模态扩展：LVLM的不确定性

### 4.1 多模态引入的额外不确定性

```
LVLM不确定性来源
├── 文本输入不确定性（与LLM相同）
├── 视觉输入不确定性（新增）
│   ├── 图像质量与分辨率
│   ├── 视觉歧义（模糊图像）
│   └── 跨模态对齐误差
├── 跨模态融合不确定性（新增）
│   ├── 文本-视觉不一致
│   └── 模态权重失衡
└── 生成不确定性
    ├── Token级（同LLM）
    └── 语义级（同LLM，但更复杂）
```

### 4.2 多模态对语义不确定性的影响

| 影响因素 | 说明 | 示例 |
|---------|------|------|
| 图像歧义 | 同一图像可被不同解读 | 医学影像的不同诊断 |
| 跨模态矛盾 | 文本与图像信息冲突 | 图中是猫但描述说狗 |
| 视觉幻觉 | 对图像内容的错误描述 | 描述图像中不存在的物体 |

## 五、关键见解与总结

### 5.1 核心论点

1. **Token级不确定性对LLM根本不充分**：开放世界中Token概率≠语义可靠性
2. **语义级不确定性是必要的范式转变**：从"模型对Token的选择"转向"模型对含义的把握"
3. **超越幻觉框架**：可靠性不只是"是否幻觉"，还包括更广泛的不确定性量化
4. **多模态扩展**：视觉输入引入额外不确定性源，需语义级方法统一处理

### 5.2 对研究方向的指引

| 方向 | 说明 |
|------|------|
| 高效语义不确定性 | 降低多次生成和蕴含判断的计算成本 |
| 语义级校准基准 | 专门针对语义不确定性的评估基准 |
| 语义不确定性 + RLHF | 在对齐过程中集成语义级不确定性信号 |
| 多模态语义不确定性 | 扩展到视觉-语言场景的统一框架 |

### 5.3 局限性

- 作为位置论文，实验验证有限，主要依赖已有方法
- 语义熵依赖外部蕴含模型，引入额外不确定性和计算开销
- Token级与语义级不确定性的关系缺乏形式化量化
- 语义级方法在生产系统中的实际部署挑战未充分讨论
- 当前方法主要适用于短答案任务，对长文本生成泛化性未知

### 5.4 定量结论总结

| 指标 | Token级 (Perplexity) | 语义级 (Semantic Entropy) | 改善幅度 |
|------|---------------------|-------------------------|---------|
| ECE | 31.50% | 13.53% | **-57%** |
| AUROC | 54.07% | 80.94% | **+27pp** |
