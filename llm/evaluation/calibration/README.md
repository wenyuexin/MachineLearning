# LLM 校准与不确定性量化专题

**主题**: 大语言模型（LLM）的校准与不确定性量化研究

**创建日期**: 2026-04-28

---

## 概述

LLM 的校准面临独特挑战：
- **生成任务**：答案形式多样，难以定义"正确"
- **序列预测**：需要序列级置信度度量
- **黑盒访问**：API 调用无法获取内部概率
- **RLHF 对齐**：对齐后校准性能可能退化

本专题收集 LLM 校准领域核心论文，涵盖不确定性量化、校准度量变体、以及实践方法。

---

## LLM特有的校准挑战

### 与传统分类任务的对比

| 维度 | 传统分类 | LLM生成 |
|------|----------|---------|
| 输出空间 | 固定类别 | 开放词汇 |
| 正确判定 | 精确匹配 | 语义相似 |
| 置信度来源 | Softmax最大值 | 序列概率/采样 |
| ECE适用性 | 直接适用 | 需要变体 |

### 关键问题

1. **如何定义"正确"？**
   - 多种表达方式语义等价
   - 部分正确如何评价

2. **如何获取置信度？**
   - 白盒：访问token概率
   - 黑盒：仅通过采样推断

3. **RLHF如何影响校准？**
   - 对齐可能损害校准
   - 如何在保持对齐的同时改善校准

## 关键概念速查

| 概念 | 说明 |
|------|------|
| **Flex-ECE** | 使用语义相似度替代精确匹配的ECE变体 |
| **Semantic Entropy** | 基于语义聚类计算的熵，考虑语义等价 |
| **Self-Consistency** | 多次采样取多数，一致性反映置信度 |
| **Token Probability** | 生成token的概率，白盒方法基础 |
| **Hallucination Detection** | 通过低置信度检测幻觉 |
| **Conformal Prediction** | 提供统计保证的预测区间 |
