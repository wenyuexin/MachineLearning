# 深度学习基础 (Deep Learning)

## 目录结构

```
deep-learning/
├── architectures/             # 基础架构
│   # CNN、RNN、Transformer等经典网络结构
│
├── foundations/               # 训练技巧与优化方法
```

- `architectures/` — 掌握CNN、RNN、Transformer等经典架构
- `foundations/` — 理解训练技巧（优化器、正则化、初始化）

**进阶阶段**
- `generative/` — 学习生成模型（GAN、VAE、Diffusion）
