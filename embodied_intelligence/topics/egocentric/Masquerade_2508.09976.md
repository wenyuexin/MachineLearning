# Masquerade：使用数据编辑从野外人类视频学习

**论文信息**
- 论文标题：Masquerade: Learning from In-the-wild Human Videos using Data-Editing
- 中文标题：Masquerade：使用数据编辑从野外人类视频学习
- 作者：M. Lepert, J. Fang, J. Bohg
- 机构：Stanford University
- arXiv: [2508.09976](https://arxiv.org/abs/2508.09976)

---

## 一、论文整体思路

### 1.1 研究背景

野外人类视频（in-the-wild）数量庞大但难以直接使用：
- 缺乏动作标签
- 视觉形态差距（人手 vs 机器人）
- 环境差异大

### 1.2 核心创新

Masquerade提出**视频编辑**方法，将人类视频"机器人化"，然后直接用于机器人策略训练。

**核心思路**：
```
人类视频 → 编辑(人手→机器人) → 机器人化演示 → 训练策略
```

## 二、方法架构

### 2.1 视频编辑流程

```
┌───────────────────────────────────────────────────────┐
│               Masquerade 视频编辑流程                 │
├───────────────────────────────────────────────────────┤
│                                                       │
│  原始人类视频                                         │
│  ┌─────────────────────────────────────────┐         │
│  │ 第一人称视角，包含人手操作               │         │
│  └─────────────────────────────────────────┘         │
│                        │                              │
│                        ▼                              │
│  步骤1: 3D手部姿态估计                                │
│  ┌─────────────────────────────────────────┐         │
│  │ 检测手部关键点，估计3D位姿               │         │
│  └─────────────────────────────────────────┘         │
│                        │                              │
│                        ▼                              │
│  步骤2: 人手移除 (Inpainting)                         │
│  ┌─────────────────────────────────────────┐         │
│  │ 将人手区域擦除，填充背景                 │         │
│  └─────────────────────────────────────────┘         │
│                        │                              │
│                        ▼                              │
│  步骤3: 机器人叠加渲染                                │
│  ┌─────────────────────────────────────────┐         │
│  │ 渲染双臂机器人，跟踪末端执行器轨迹       │         │
│  └─────────────────────────────────────────┘         │
│                        │                              │
│                        ▼                              │
│  机器人化演示视频                                     │
│  ┌─────────────────────────────────────────┐         │
│  │ 可直接用于策略训练                       │         │
│  └─────────────────────────────────────────┘         │
│                                                       │
└───────────────────────────────────────────────────────┘
```

### 2.2 关键技术

| 技术 | 描述 |
|------|------|
| 手部检测 | 估计3D手部姿态 |
| Inpainting | 移除人手，填充背景 |
| 机器人渲染 | 叠加渲染机器人模型 |
| 轨迹跟踪 | 确保动作一致性 |

## 三、实验与结果

### 3.1 数据规模

- 675K帧编辑视频
- 来自野外人类视频
- 无需专门采集

### 3.2 主要结果

- 可直接从编辑视频训练策略
- 避免了人-机器人形态差距
- 支持多样化任务
