# GR00T N1：通用人形机器人开源基础模型

**论文信息**
- 论文标题：GR00T N1: An Open Foundation Model for Generalist Humanoid Robots
- 中文标题：GR00T N1：通用人形机器人开源基础模型
- 作者：Johan Bjorck, Fernando Castañeda, Nikita Cherniadev, Xingye Da, Runyu Ding, Linxi Fan, Yu Fang, Dieter Fox, Fengyuan Hu, Spencer Huang 等 (NVIDIA Team)
- 机构：NVIDIA
- arXiv: [2503.14734](https://arxiv.org/abs/2503.14734)
- 项目主页: https://research.nvidia.com/labs/gear/

> **核心创新**：双系统架构 + 三层数据预训练 - 592.9M轨迹/8,375.7小时数据，开源人形机器人VLA基础模型

---

## 一、论文整体思路

### 1.1 研究背景

通用机器人需要一个多功能身体和一个智能大脑。近期人形机器人的进展显示了作为在人类世界中构建通用自主能力的硬件平台的巨大前景。

**核心问题**：
- 人形机器人数据极少
- 需要大规模预训练来获得泛化能力
- 传统机器人数据集不适用于人形

### 1.2 核心创新

GR00T N1是NVIDIA推出的**开源通用人形机器人基础模型**，采用**双系统架构**和**三层数据预训练**策略。

**核心特点**：
- Vision-Language-Action (VLA) 模型
- 跨形态支持：从桌面机械臂到灵巧人形机器人
- 双系统架构（System 2推理 + System 1动作）
- 开源模型权重

## 二、方法架构

### 2.1 双系统架构

受人类认知处理启发（Kahneman, 2011）：

```
┌─────────────────────────────────────────────────────────┐
│                GR00T N1 双系统架构                       │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  System 2: 推理模块 (VLM)                               │
│  ┌─────────────────────────────────────────────┐       │
│  │ • 预训练Vision-Language Model               │       │
│  │ • 处理视觉感知和语言指令                     │       │
│  │ • 理解环境和任务目标                         │       │
│  │ • 运行频率：10Hz (NVIDIA L40 GPU)           │       │
│  └─────────────────────────────────────────────┘       │
│                        │                                │
│                        ▼                                │
│  System 1: 动作模块 (Diffusion Transformer)            │
│  ┌─────────────────────────────────────────────┐       │
│  │ • 使用Action Flow-Matching训练              │       │
│  │ • Cross-attend到VLM输出tokens               │       │
│  │ • 实时生成流畅的电机动作                      │       │
│  └─────────────────────────────────────────────┘       │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 2.3 数据规模统计

| 数据类型 | 轨迹数 | 小时数 |
|----------|--------|--------|
| 仿真数据 | 125.5M | 1,742.6 |
| 神经数据 | 23.8M | 827.3 |
| **总计** | **592.9M** | **8,375.7** |

### 2.4 神经轨迹生成

GR00T使用生成模型创建仿真数据：
- 微调图像到视频模型
- 生成反事实轨迹
- 支持新语言提示

## 三、训练流程

### 3.1 端到端跨三层预训练

1. **视频数据**：学习动作先验和语义理解
2. **仿真数据**：学习场景-动作关联
3. **机器人数据**：学习真实执行能力

### 3.2 数据处理

| 数据类型 | 处理方式 |
|----------|----------|
| 视频 | 潜在动作标注 + IDM伪动作 |
| 仿真 | 直接使用动作标签 |
| 机器人 | 遥操作动作标签 |

## 四、能力与特点

### 4.1 支持的能力

**GR00T N1 支持的能力**：
1. 多任务操作
2. 零样本泛化到新任务
3. 新物体识别和操作
4. 语言条件控制
5. 跨形态支持

### 4.2 跨形态支持

- 桌面机械臂
- 灵巧人形机器人
- 统一的模型架构

## 六、关键贡献

### 6.1 模型贡献

1. **首个开源人形VLA**：为人形机器人社区提供基础模型
2. **双系统架构**：System 2推理 + System 1动作
3. **三层数据训练**：系统性利用视频、仿真、机器人数据
4. **流匹配架构**：高效的连续动作生成

### 6.2 数据贡献

1. GR00T N1 人形预训练数据集
2. 多样化任务覆盖
3. 标准化数据格式
4. 最大规模：592.9M轨迹

### 6.3 开源贡献

- 模型权重开源
- 训练代码开源
- 数据处理工具

## 七、技术细节

### 7.1 流匹配（Flow Matching）

不同于扩散模型，流匹配：
- 更平滑的动作生成
- 高效采样
- 确定性轨迹学习

### 7.2 潜在动作学习

从无动作标签的视频中学习：
- 视频帧间编码
- 潜在动作表示
- 迁移到真实动作

### 7.3 遥操作系统

支持多种遥操作设备：
- VIVE Ultimate Tracker：腕部追踪
- Xsens Metagloves：手指追踪
- Apple Vision Pro：替代方案
- Leap Motion：手势追踪

---

## 八、局限性与未来方向

### 8.1 局限性

1. **数据规模**：仍需更多数据
2. **任务覆盖**：当前任务种类有限
3. **精细操作**：高精度任务仍有挑战

### 8.2 未来方向

1. 更大规模数据收集
2. 更多机器人形态支持
3. 强化学习后训练
4. 实时推理优化
