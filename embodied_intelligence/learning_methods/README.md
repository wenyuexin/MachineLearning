# 学习方法

本目录整理具身AI的学习方法和训练范式。

## 学习方法概览

| 方法 | 核心思想 | 数据需求 | 优势 | 劣势 |
|------|----------|----------|------|------|
| **模仿学习** | 从专家演示学习 | 高质量演示数据 | 快速获取策略 | 泛化能力有限 |
| **强化学习** | 通过试错优化策略 | 环境交互 | 突破性能上限 | 样本效率低 |
| **迁移学习** | 跨域知识迁移 | 源域+目标域数据 | 减少学习成本 | 负迁移风险 |
| **元学习** | 学会如何学习 | 多任务数据 | 快速适应新任务 | 计算开销大 |

## 1. 模仿学习（Imitation Learning）

### 核心概念

让智能体通过模仿专家的行为来学习技能。

**数学定义**：
$$\min_\pi \mathbb{E}_{s \sim D}[d(\pi(a|s), \pi^*(a|s))]$$

### 行为克隆（Behavior Cloning）

最简单的模仿学习方法，形式化为监督学习：

$$L(\pi) = -\mathbb{E}_{\tau \sim D}[\log \pi(a|s)]$$

### 数据采集方式

| 方式 | 描述 | 优点 | 缺点 |
|------|------|------|------|
| 遥操作 | VR手柄、力反馈设备控制 | 数据质量高 | 需要设备 |
| 动捕系统 | 记录人类动作映射到机械臂 | 自然的操作方式 | 设备昂贵 |
| 直接示教 | 人工移动机械臂 | 简单直接 | 精度受限 |
| 事后标注 | 从Planner生成轨迹筛选 | 可扩展 | 需要筛选机制 |

### 策略网络构建

| 架构 | 特点 | 代表工作 |
|------|------|----------|
| MLP | 简单，适用于低维空间 | 早期方法 |
| ACT | Action Chunking，时序一致性 | ACT (2023) |
| Diffusion Policy | 多模态分布建模，高质量生成 | Diffusion Policy (2023) |
| Transformer策略 | 序列建模，长程依赖 | RT系列 |
| LLM策略 | 预训练知识，推理能力 | EmbodiedGPT |

### 挑战

- **分布偏移（Distribution Shift）**: 训练与测试分布不一致
- **数据效率低**: 需要大量演示
- **缺乏错误恢复**: 无法处理未见过的状态

## 2. 强化学习（Reinforcement Learning）

### 核心概念

通过与环境的迭代交互来优化策略。

### 主要算法

| 算法 | 类型 | 特点 |
|------|------|------|
| PPO | On-policy | 稳定，易于实现 |
| SAC | Off-policy | 样本效率高 |
| DQN | Value-based | 离散动作空间 |
| QT-Opt | Off-policy | 大规模自监督RL |

### 大模型赋能

| 应用方向 | 方法 |
|----------|------|
| 奖励函数设计 | LLM自动生成奖励函数、VLM视觉奖励 |
| 策略网络构建 | 扩散策略、Transformer策略、LLM策略 |

### RLHF系列

| 方法 | 特点 |
|------|------|
| RLHF | 基于人类反馈的强化学习 |
| DPO | 直接偏好优化，无需奖励模型 |
| GRPO | 群组相对策略优化 |

### 挑战

- **样本效率低**: 需要大量交互
- **奖励设计困难**: 稀疏奖励问题
- **安全风险**: 真实环境探索危险

## 3. 迁移学习（Transfer Learning）

### 主要场景

| 场景 | 描述 | 关键技术 |
|------|------|----------|
| Sim-to-Real | 仿真到真实迁移 | 域随机化、域适应 |
| 跨任务迁移 | 任务间能力迁移 | 多任务学习、元学习 |
| 跨形态迁移 | 不同机器人平台 | 统一表示学习 |
| 跨场景迁移 | 不同环境适应 | 场景泛化 |

### Sim-to-Real方法

| 方法 | 描述 |
|------|------|
| 域随机化 | 随机化光照、纹理、物理参数 |
| 域适应 | 学习仿真与真实之间的映射 |
| 渐进迁移 | 逐步从仿真过渡到真实 |
| 离线预训练 | PTR, V-PTR |

### 挑战

- **仿真-真实差距**: 物理、视觉、动力学差异
- **负迁移**: 源域知识干扰目标域
- **领域偏移**: 分布不一致

## 4. 元学习（Meta-Learning）

### 核心思想

学习"如何学习"，快速适应新任务。

### 方法分类

| 类型 | 核心思想 | 代表工作 |
|------|----------|----------|
| 基于优化 | 学习好的初始化参数 | MAML |
| 基于记忆 | 利用记忆模块存储经验 | Meta-SGD |
| 基于度量 | 学习任务间相似性度量 | Prototypical Networks |

### 应用场景

- 快速适应新任务（少样本学习）
- 跨平台泛化
- 在线持续学习

## 5. 后训练方法

### 监督微调（SFT）

适应特定机器人/任务，使用专家演示数据。

### 强化学习微调（RLFT）

突破性能上限，通过环境交互优化。

### 推理扩展（Inference-time Scaling）

执行前多步推理，提高决策准确性。

## 方法选择建议

| 场景 | 推荐方法 |
|------|----------|
| 数据充足、任务固定 | 模仿学习 |
| 动态环境、有明确目标 | 强化学习 |
| 有相关任务经验可复用 | 迁移学习 |
| 需要快速适应新任务 | 元学习 |
| 混合策略 | 先模仿学习，后强化学习优化 |
